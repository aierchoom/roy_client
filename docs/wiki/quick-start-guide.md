## 5 分钟快速上手

### 第一步：安装应用

根据您的平台下载并安装 SecretRoy：

| 平台 | 下载方式 |
|------|----------|
| Android | 下载 APK 安装 |
| iOS | App Store 搜索 "SecretRoy" |
| Windows | 下载安装包 |
| macOS | 下载 DMG 安装 |
| Linux | Snap 或下载 AppImage |

### 第二步：创建主密码

首次启动时，系统会要求您创建一个**主密码**。

> ⚠️ **重要**：主密码用于加密您的所有数据，请务必牢记！主密码丢失后数据无法恢复。

**设置强密码建议**：
- 至少 12 个字符
- 包含大小写字母、数字和符号
- 避免使用生日、姓名等易猜信息

### 第三步：添加第一个账户

1. 点击右下角的 **+** 按钮
2. 选择 **网站模板**
3. 填写账户信息：
   - 账户名称：如 "GitHub"
   - 用户名：您的 GitHub 用户名
   - 密码：您的 GitHub 密码
   - 网址：https://github.com（可选）
4. 点击 **保存**

### 第四步：查看和管理账户

- 点击账户卡片查看详情
- 点击复制图标复制密码
- 点击编辑按钮修改账户信息

### 第五步：设置同步（可选）

如需在多设备间同步数据：

1. 进入 **设置** → **数据同步**
2. 输入同步服务器地址
3. 点击 **连接**
4. 按提示完成配对

## 核心功能速览

### 账户管理

| 操作 | 说明 |
|------|------|
| 添加账户 | 点击 + 按钮，选择模板并填写信息 |
| 编辑账户 | 点击账户 → 编辑按钮 |
| 删除账户 | 点击账户 → 删除按钮 |
| 搜索账户 | 在搜索框输入关键词 |
| 复制密码 | 在账户详情页点击复制图标 |

### 模板系统

模板定义了账户的结构：

| 内置模板 | 包含字段 |
|----------|----------|
| 网站模板 | 网站、账号、密码、备注 |

**创建自定义模板**：
1. 进入"模板"标签页
2. 点击 + 新建模板
3. 设置模板名称和字段
4. 保存

### 同步功能

SecretRoy 支持四种同步方式：

| 方式 | 适用场景 |
|------|----------|
| 云服务器 | 多设备自动同步（需 vault-level token 认证） |
| 面对面链接 | 同一可信局域网内加新设备 |
| 远程配对 | 不在同一局域网时由可信设备批准 |
| 离线恢复码 | 无法配对时手动恢复密钥或数据快照 |

### 2FA/TOTP 验证码

- 进入 **账户** → **2FA** 标签页管理独立 TOTP 凭据
- 支持移动端相机扫码导入、桌面端粘贴二维码图片导入
- TOTP 凭据与账号解耦，通过模板 2FA 字段进行关联
- 验证码复制后 45 秒自动从剪贴板清理

## 常用操作指南

### 保护敏感字段

在账户详情页，保密字段（如密码）默认隐藏显示：
- 点击 👁 图标查看明文
- 点击 📋 图标复制到剪贴板

### 使用生物识别解锁

1. 进入 **设置** → **安全设置**
2. 开启 **生物识别解锁**
3. 首次需要验证主密码
4. 之后可使用指纹或面容快速解锁

### 设置自动锁定

1. 进入 **设置** → **安全设置**
2. 选择 **自动锁定时间**
3. 可选：立即、1分钟、5分钟、15分钟

### 导出备份

1. 确保已连接同步服务器
2. 点击 **立即同步**
3. 数据将自动备份到服务器

## 安全建议

### 密码安全

✅ **推荐做法**：
- 使用强主密码
- 启用生物识别解锁
- 定期更换重要密码
- 使用密码生成器

❌ **避免做法**：
- 使用简单密码（如 123456）
- 多个账户使用相同密码
- 将密码写在纸上或文件中

### 数据安全

✅ **推荐做法**：
- 定期同步备份
- 记住主密码
- 不在公共设备上保存密码
- 定期检查账户安全

### 同步安全

- 服务器只存储加密数据
- 只有您的主密码能解密
- 使用 HTTPS 加密传输
- 定期检查同步设备列表
