## 1. 产品简介

### 1.1 什么是 SecretRoy？

SecretRoy 是一款**分布式密码管理器**，专注于安全、隐私和多设备同步。与传统云端密码管理器不同，SecretRoy 采用**本地优先**架构，您的数据始终在您的设备上加密存储。

### 1.2 核心特性

| 特性 | 说明 |
|------|------|
| 🔐 **端到端加密** | 所有数据在本地加密，服务器无法解密 |
| 🔄 **CRDT 同步** | 支持离线编辑，自动合并多设备更改 |
| 📱 **跨平台** | 支持 Android、iOS、Windows、macOS、Linux |
| 🎨 **自定义模板** | 灵活定义不同类型的账户结构 |
| 🌐 **多种同步方式** | 云服务器、面对面链接、远程配对、离线恢复码 |

### 1.3 适用场景

- 🔑 个人密码管理
- 🏢 团队共享账户信息
- 🔒 敏感信息加密存储
- 📋 恢复码、API Key、许可证等敏感信息管理

## 2. 快速入门

## 3. 账户管理

### 3.1 添加账户

1. 点击右下角的 **+** 按钮
2. 选择账户模板（默认为"网站模板"）
3. 填写账户信息：
   - **账户名称**：如"淘宝账户"、"公司邮箱"
   - **关联邮箱/备注**：可选的附加信息
   - **模板字段**：根据选择的模板填写

4. 点击 **保存** 完成添加

### 3.2 编辑账户

1. 在账户列表中点击要编辑的账户
2. 点击右上角的 **编辑** 按钮
3. 修改需要更新的字段
4. 点击 **保存** 确认修改

### 3.3 删除账户

1. 在账户详情页点击 **删除** 按钮
2. 确认删除操作

> ⚠️ **注意**: 删除操作不可撤销，请谨慎操作

### 3.4 搜索账户

- 在账户列表顶部使用搜索框
- 支持搜索账户名称、邮箱等可搜索字段

### 3.5 复制密码

1. 在账户详情页找到密码字段
2. 点击复制图标
3. 密码将复制到剪贴板

> 💡 **提示**: 保密字段默认隐藏显示，点击眼睛图标可查看明文

## 4. 模板系统

### 4.1 什么是模板？

模板定义了账户的结构和字段。每个账户都基于一个模板创建，模板决定了该账户包含哪些字段、字段的类型和属性。

### 4.2 内置模板

SecretRoy 提供以下内置模板：

| 模板名称 | 包含字段 |
|----------|----------|
| **网站模板** | 网站、账号、密码、备注 |

### 4.3 创建自定义模板

1. 进入 **模板** 标签页
2. 点击 **+ 新建模板** 按钮
3. 填写模板基本信息：
   - **模板名称**：如"会员卡"
   - **副标题**：简要描述模板用途

4. 添加字段：
   - 点击 **添加字段**
   - 设置字段名称、类型、属性

5. 点击 **保存模板** 完成

### 4.4 字段类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 文本 | 普通文本输入 | 用户名 |
| 密码 | 加密显示的敏感文本 | 密码 |
| 数字 | 纯数字输入 | 卡号 |
| 邮箱 | 邮箱格式验证 | user@example.com |
| 电话 | 电话号码 | 138 0000 0000 |
| 网址 | URL 格式 | https://example.com |
| 时间 | 日期时间选择 | 有效期 |
| 自定义 | 自定义格式 | 其他 |

### 4.5 字段属性

| 属性 | 说明 |
|------|------|
| **必填** | 创建账户时必须填写 |
| **保密** | 默认隐藏显示，点击可查看 |
| **允许编辑** | 账户创建后是否可修改 |
| **可搜索** | 是否可通过搜索找到 |
| **允许复制** | 是否可复制到剪贴板 |
| **主字段** | 标记为主要识别字段 |

### 4.6 编辑模板

1. 在模板列表中点击要编辑的模板
2. 修改模板信息或字段
3. 点击 **保存** 确认修改

> ⚠️ **注意**: 已有账户使用的模板字段标识不可修改，避免影响已保存的数据

## 5. 同步功能

### 5.1 同步概述

SecretRoy 支持**多设备同步**，采用 CRDT（无冲突复制数据类型）技术，即使设备离线编辑也能自动合并更改。

### 5.2 同步方式

#### 方式一：云服务器同步

将数据同步到自建的 SecretRoy 服务器。

**配置步骤**：
1. 进入 **设置** → **数据同步**
2. 点击 **服务器设置**
3. 输入服务器地址（如 `https://sync.example.com`）
4. 点击 **连接** 按钮
5. 首次连接需要配对

#### 方式二：面对面链接

在同一可信局域网内，通过 8 位临时码把新设备加入同一个 vault。配对码窗口关闭后，本次密钥包立即不可领取。

**主机端（桌面设备）**：
1. 进入 **设置** → **数据同步**
2. 在 **密钥恢复与设备链接** 中点击 **显示临时码**
3. 保持弹窗打开，等待新设备输入

**加入端（手机或其他设备）**：
1. 进入 **设置** → **数据同步**
2. 在 **面对面链接** 中点击 **输入临时码**
3. 输入主机弹窗中显示的 8 位临时码
4. 确认链接并导入

#### 方式三：远程配对

如果两台设备都能访问同一个同步服务器，但不在同一局域网，可以使用服务器中转配对：

1. 已有设备创建远程配对会话并显示配对码
2. 新设备输入远程配对码并提交加入请求
3. 已有设备检查请求并手动批准
4. 新设备检查授权结果并导入 vault 密钥
5. 新设备执行一次同步以拉取已有数据

#### 方式四：离线恢复码

当无法使用面对面链接或远程配对时，可以使用提前导出的离线恢复码。恢复码使用 `sroy-recovery:` 协议格式，并由恢复密码加密。

1. 在已有设备上导出离线恢复码
2. 将恢复码和恢复密码分开保存
2. 进入 **设置** → **数据同步**
3. 在 **离线恢复码** 中点击 **导入恢复码**
4. 粘贴恢复码并输入恢复密码
5. 确认导入；如果本机已有数据，需要明确确认覆盖

> 注意：密钥同步只共享 vault 身份，接收设备会保留自己的设备 ID。`sroy-link:` 是内部兼容码，不建议用户手动保存或粘贴。
>
> 新 vault 首次连接服务器时会自动获得一个 vault-level API token，后续同步需在请求中携带该 token。token 会随配对包安全传递到新设备。

### 5.3 同步状态

| 状态 | 说明 |
|------|------|
| 🟢 **已就绪 (idle)** | 同步正常，本地与远端一致 |
| 🔵 **连接中 (connecting)** | 正在与同步服务器建立连接 |
| 🔵 **拉取中 (pulling)** | 正在从服务器拉取远端更新 |
| 🔵 **推送中 (pushing)** | 正在将本地变更推送到服务器 |
| 🟡 **恢复中 (conflictRecovery)** | 正在处理冲突恢复 |
| 🔴 **网络不可达 (networkUnreachable)** | 无法连接到服务器，请检查网络 |
| 🔴 **服务端错误 (serverError)** | 服务器暂时不可用 |
| 🔴 **认证错误 (authError)** | 服务器拒绝访问，请检查 vault token 或重新配对 |
| 🔴 **协议错误 (protocolError)** | 同步数据解析或校验失败 |
| ⚫ **离线 (offline)** | 未连接到同步服务器 |

### 5.4 冲突处理

当多个设备同时修改同一账户时，SecretRoy 会自动检测并处理冲突：

1. **自动合并** - 如果修改的是不同字段，自动合并
2. **冲突标记** - 如果修改的是同一字段，标记为冲突
3. **人工审阅** - 冲突条目会在收件箱中显示，需要人工确认保留哪个版本

### 5.5 手动同步

点击同步状态卡片上的 **立即同步** 按钮，手动触发一次同步。

## 6. 安全设置

### 6.1 主密码

主密码用于加密您的所有数据。

**修改主密码**：
1. 进入 **设置** → **安全设置**
2. 点击 **修改主密码**
3. 输入当前密码和新密码
4. 确认修改

> ⚠️ **重要**: 主密码丢失将无法恢复数据！

### 6.2 生物识别

启用指纹或面容识别快速解锁。

1. 进入 **设置** → **安全设置**
2. 开启 **生物识别解锁**
3. 首次需要验证主密码
4. 之后可使用生物识别快速解锁

### 6.3 自动锁定

设置应用自动锁定时间：

1. 进入 **设置** → **安全设置**
2. 选择 **自动锁定时间**
3. 可选：立即、1分钟、5分钟、15分钟、从不

### 6.4 剪贴板清理

应用会自动清理复制到系统剪贴板的敏感数据，避免密码、配对码、恢复码和 TOTP 验证码长期暴露：

- **高风险内容**（密码、配对码、恢复码、TOTP 验证码）：默认 45 秒后自动清理。
- 清理前会检查剪贴板当前内容是否仍是你复制的那条，避免误删你后续复制的其他内容。

此功能无需手动开启，复制敏感内容时自动生效。

## 7. 常见问题

### Q1: 忘记主密码怎么办？

**答**: 很遗憾，SecretRoy 采用端到端加密，主密码丢失后数据无法恢复。建议您：
- 设置密码提示
- 使用密码管理器保存主密码
- 定期备份加密数据

### Q2: 同步失败怎么办？

**答**: 请检查：
1. 服务器地址是否正确
2. 网络连接是否正常
3. 服务器是否运行
4. 查看错误提示信息

如使用 `localhost` 或 `127.0.0.1`，手机端请改用电脑的局域网 IP。

### Q3: 如何迁移到新设备？

**答**: 有两种方式：
1. **同步迁移**: 在新设备上登录同一同步账号
2. **手动迁移**: 导出旧设备数据，在新设备导入

### Q4: 数据存储在哪里？

**答**: 所有数据以加密形式存储在本地设备：
- Android: 加密存储在应用私有目录
- iOS: 存储在 Keychain 和应用沙盒
- 桌面: 存储在用户配置目录

### Q5: 服务器能看到我的数据吗？

**答**: 不能。SecretRoy 采用端到端加密：
- 数据在本地加密后才上传
- 服务器只存储加密后的密文
- 只有您的主密码能解密数据

### Q6: 为什么选择 CRDT 同步？

**答**: CRDT 的优势：
- 支持离线编辑
- 自动合并冲突
- 无需中央协调
- 最终一致性保证

### Q7: 如何备份数据？

**答**:
1. **云同步备份**: 启用同步功能，数据自动备份到服务器
2. **本地备份**: 设置中导出加密备份文件

### Q8: 应用支持哪些平台？

**答**: SecretRoy 支持：
- Android 5.0+
- iOS 12.0+
- Windows 10+
- macOS 10.14+
- Linux (主流发行版)
